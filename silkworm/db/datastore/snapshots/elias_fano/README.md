Elias-Fano encoding is a high bits / low bits representation of a monotonically increasing sequence of N > 0 natural numbers x[i]
0 <= x[0] <= x[1] <= ... <= x[N-2] <= x[N-1] <= U
where U > 0 is an upper bound on the last value.
